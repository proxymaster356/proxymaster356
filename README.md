### 🌾 CropIntel
[![Status](https://img.shields.io/badge/Status-✅%20Completed-success?style=for-the-badge&labelColor=1a1a2e)](https://github.com/proxymaster356)
[![Type](https://img.shields.io/badge/Type-Android%20%2B%20AI%20%2B%20IoT-blue?style=for-the-badge&labelColor=1a1a2e)](https://github.com/proxymaster356)

</div>

> 🏆 **Real-time Android app for intelligent crop management**

A full-stack agricultural intelligence platform that fuses live IoT sensor data (soil moisture, temperature, humidity, pH) with real-time weather API feeds to give farmers actionable insights. Features an on-device **plant disease detection model** using computer vision, and an AI recommendation engine for optimizing crop yield based on field conditions, growth stage, and historical patterns.

| Feature | Details |
|:--------|:--------|
| 📡 **Sensor Integration** | Live data ingestion via ESP32/Arduino over MQTT/Firebase |
| 🌦️ **Weather AI** | Real-time weather integration for irrigation and pest risk alerts |
| 🍃 **Disease Detection** | Plant disease from leaf images using fine-tuned CNN |
| 🤖 **Yield Optimizer** | Personalized crop management recommendations |
| 📱 **Native Android** | Built with Kotlin + Jetpack Compose |
